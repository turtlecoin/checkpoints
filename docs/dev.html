  For Developers

Getting Latest Checkpoints IPFS Hash via JSON

  1. Pull the JSON response from
  2. The response will be a JSON object like shown below:
    { "hash": "QmSLT3JNFkZMyjaEDpy4kDELqx6NdpJUB5aBjpP1N1sLh2" }
  3. Reference the hash property of the object in your application.

Getting Latest Checkpoints IPFS Hash via DNS

  1. Query the DNS TXT record for: _dnslink.checkpoints.turtlecoin.dev
  2. The TXT record will contain a value like shown below
    dnslink=/ipfs/QmSLT3JNFkZMyjaEDpy4kDELqx6NdpJUB5aBjpP1N1sLh2
  3. Cut apart the result such you are left with the hash. (ex. QmSLT3JNFkZMyjaEDpy4kDELqx6NdpJUB5aBjpP1N1sLh2)
