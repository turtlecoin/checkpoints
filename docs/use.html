  How to Use

Obtaining The Latest TurtleCoin® Software

  1. Proceed on over to TurtleCoin® Releases to grab the latest copy for your system.

Obtaining TurtleCoin® Checkpoints - Browser

  1. Right click one of the links on the Checkpoints page, and select Save link as...
  2. Save the file with a name that is easy to remember. (ie. checkpoints.csv)

Obtaining TurtleCoin® Checkpoints - CLI Mode

  1. Copy one of the links on the Checkpoints page
  2. Download the checkpoints via curl:
      curl <link> -o checkpoints.csv

      curl -o checkpoints.csv

Using the Checkpoints - Windows

  1. Copy the checkpoints you downloaded into the same folder as your copy of TurtleCoind.exe
  2. Open a command prompt window in the folder where you just placed the checkpoints file.

      ProTip: Type cmd in Windows Explorer when in the folder to start command prompt from that folder.

  3. In the command prompt window, type the following to start the daemon with the checkpoints:

      TurtleCoind.exe --load-checkpoints checkpoints.csv

Using the Checkpoints - Linux / Apple

  1. Copy the checkpoints you downloaded into the same folder as your copy of TurtleCoind
  2. Open a terminal window in the folder where you just placed the checkpoints file.

      ProTip: Your file manager may provide the ability to open a terminal in your current directory. Right-click in the folder to see if the option is available.

  3. In the terminal window, type the following to start the daemon with the checkpoints:

      ./TurtleCoind --load-checkpoints checkpoints.csv
